# LythMono (Personal Fork)

This is a personal fork of [`why-trv/LythMono`](https://github.com/why-trv/LythMono).

The motive for the fork is simple: I wanted a version of the font tuned to my own preferences, especially around the bundled Nerd Font variants and individual glyph details like the `7`.

This repo is mainly for building and releasing those custom font artifacts. It is not meant to replace the upstream project or its documentation.

For the original project, full documentation, screenshots, and build notes, use:

- [why-trv/LythMono](https://github.com/why-trv/LythMono)

The original upstream-style README from this fork has been preserved as [README_original.md](./README_original.md).
